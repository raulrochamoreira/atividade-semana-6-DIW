## Descrição do Projeto
Refatoração da home-page criada na etapa anterior, substituindo a abordagem de responsividade manual com CSS puro (media queries, flexbox, grid) pela utilização do framework **Bootstrap 5**.

### Ferramentas e Tecnologias
- HTML5 e CSS3
- Bootstrap 5 (via CDN)
- Editor de Código (VS Code)
- Git e GitHub

### Elementos Implementados com Bootstrap
- **Cabeçalho/Header:** Utilização de `navbar` com `navbar-toggler` e collapse para a versão mobile.
- **Main Section:** Uso de `container-fluid`, `img-fluid` e formatação de textos utilitários (`display-4`, `lead`).
- **Cards (Serviços):** Uso do sistema de grid (`row`, `col-md-4`) e do componente `card` (`card-img-top`, `card-body`) para manter a uniformidade e responsividade.
- **Footer:** Espaçamento e tipografia gerenciados por classes utilitárias (`bg-dark`, `text-center`, `py-4`).
